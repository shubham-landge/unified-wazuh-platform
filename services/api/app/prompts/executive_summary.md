# Executive Summary Prompt
#
# Used when creating a management-friendly summary of an incident.

Create a concise executive summary of this security incident suitable for a non-technical audience.

Incident Title: {incident_title}
Severity: {severity}
Duration: {duration}
Affected Assets: {affected_assets}
MITRE Tactic: {mitre_tactic}
Status: {status}

## Executive Summary Template

**What happened**
[2-3 sentence summary]

**Impact**
[What systems/data/users affected]

**Actions taken**
[Key investigation and response steps]

**Recommendations**
[1-2 actionable recommendations]

**Status**
[Current status and next steps]

Keep the summary professional, concise, and business-focused.
Avoid technical jargon unless necessary. If technical terms are needed, explain them in parentheses.
